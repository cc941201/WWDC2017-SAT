# SAT

SAT is an iOS Swift Playground that teaches declarative programming in SAT.

It is inspired by [Declarative Methods](https://cs.jhu.edu/~jason/325/) taught by [Professor Jason Eisner](https://cs.jhu.edu/~jason/).
